**src/index.tsx:**
The plugin entry point. Must export exactly: name, slots, onActivate, onDeactivate. This is the mandatory Ship Studio plugin export shape.

Imports:
- useState, useEffect from 'react'
- STYLE_ID, PLUGIN_CSS from './styles'
- useGsd from './useGsd'
- InstallView from './views/InstallView'
- NoProjectView from './views/NoProjectView'

**useInjectStyles():** Private hook that injects CSS into document.head on mount. Checks for existing style element by STYLE_ID to ensure idempotency. Removes the style element on cleanup. Uses useEffect with empty deps array.

**ToolbarButton():** The slot component.
- State: modalOpen (boolean, default false)
- Calls useGsd() to get the gsd handle
- Calls useInjectStyles() to inject CSS
- Re-runs detection when modal opens: useEffect that calls gsd.redetect() when modalOpen transitions to true. This ensures the plugin always shows fresh state when opened.

Escape key handler: useEffect that listens for 'keydown' events when modalOpen is true. If e.key === 'Escape', close the modal. Cleanup removes the listener.

View router: switch statement on gsd.phase that selects the correct content:
- 'loading': `<div className="gsd-loading-indicator">Checking GSD status...</div>`
- 'no-project': `<NoProjectView />`
- 'gsd-not-installed': `<InstallView gsd={gsd} />`
- 'no-planning': `<div><h3>No Planning Directory</h3><p style={{ color: 'var(--text-secondary)', fontSize: 13 }}>GSD is installed but this project has no .planning/ directory. Run <code>/gsd:new-project</code> in Claude Code to start planning.</p></div>`
- 'has-planning': `<div><h3>Dashboard</h3><p style={{ color: 'var(--text-secondary)', fontSize: 13 }}>Plan dashboard coming in Phase 2.</p></div>`
- 'error': `<div className="gsd-error-state">Error: {gsd.error}</div>`

Render:
- A button with className="toolbar-icon-btn", title="GSD", onClick opens modal
- Inside the button, an SVG icon (14x14 viewBox 0 0 24 24) showing a checkmark/checklist:
  ```
  <svg width="14" height="14" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2" strokeLinecap="round" strokeLinejoin="round">
    <polyline points="9 11 12 14 22 4" />
    <path d="M21 12v7a2 2 0 0 1-2 2H5a2 2 0 0 1-2-2V5a2 2 0 0 1 2-2h11" />
  </svg>
  ```
- When modalOpen is true, render the modal overlay:
  - Outer div: className="gsd-modal-overlay", onClick closes modal
  - Inner div: className="gsd-modal", onClick={e => e.stopPropagation()} (prevent overlay click-through)
  - Header div: className="gsd-modal-header", contains "GSD" span and close button (gsd-btn gsd-btn-secondary)
  - Body div: className="gsd-modal-body", contains the view router content
