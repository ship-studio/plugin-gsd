Props: `{ showToast: (message: string, type?: 'success' | 'error') => void }`.

Component structure:

1. **Getting-started intro** (top of view): 2-3 sentences explaining what GSD is, how it works with Claude Code, and where to start. Example:
   - "Get Shit Done (GSD) is a structured planning system for Claude Code projects. It breaks work into phases with discussion, planning, execution, and verification steps. Start by running `/gsd:new-project` in Claude Code to set up your project."

2. **Core workflow loop** — vertical step-by-step flow. Each step is a styled row showing:
   - Step number (circled or bold)
   - Slash command (monospace, styled with var(--bg-tertiary) background)
   - One-liner description of what the step does
   - Click-to-copy: clicking the slash command copies it to clipboard via `navigator.clipboard.writeText(command)`, then calls `showToast('Copied /gsd:command', 'success')`.

   Core steps in order:
   1. `/gsd:new-project` — Set up a new project with requirements and roadmap
   2. `/gsd:discuss-phase` — Discuss implementation decisions for the next phase
   3. `/gsd:plan-phase` — Create executable plans with task breakdown
   4. `/gsd:execute-phase` — Run plans to build the phase
   5. `/gsd:verify-phase` — Verify all must-haves are met

3. **Utility commands section** — a secondary section with additional helpful commands:
   - `/gsd:progress` — Check overall project progress
   - `/gsd:research-phase` — Deep research before planning
   - `/gsd:debug` — Debug issues in the current phase
   - `/gsd:add-todo` — Add a todo to the project backlog

   These also have click-to-copy behavior.

4. **Visual layout**: Use `.gsd-guide-step` CSS class for each step. Use a vertical line or spacing to create a "flow" feeling. Each step should feel visually connected to the next.

CRITICAL: All slash commands must use the click-to-copy pattern. The handler:


Do NOT import or use shell/context in this component. It is purely presentational with clipboard access.

   ```typescript
   useEffect(() => {
     if (modalOpen) {
       setActiveTab(
         gsd.phase === 'gsd-not-installed' || gsd.phase === 'no-project'
   ```

   ```typescript
   const handleTabChange = (tab: 'dashboard' | 'guide') => {
     setActiveTab(tab);
     if (tab === 'guide') gsd.clearFileView();
   };
   ```
